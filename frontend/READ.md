# ShieldAI Frontend

React frontend for ShieldAI, an AI-assisted multi-layer vulnerability assessment dashboard.

## Features

- Security scan input
- File upload interface
- Vulnerability dashboard
- Risk score gauge
- Severity charts
- Risk heatmap
- AI mitigation panel
- Vulnerability table
- PDF report export
- Flask backend API integration
